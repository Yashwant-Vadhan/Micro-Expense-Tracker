# Expense Tracker Frontend (ESM + No Tailwind)

This package is ESM-ready (vite.config.mjs + "type": "module") and has no Tailwind/PostCSS.

## Quick start
1. unzip
2. cd into the folder
3. npm install --legacy-peer-deps
4. npm run dev

Make sure your backend is running and .env VITE_API_BASE points to your backend (default http://localhost:5000/api).

If install fails, run: npm cache clean --force and retry.

If you hit errors, paste node -v, npm -v, and the first 20 lines of the npm error.
